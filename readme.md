Tab bar based on [iced_aw](https://github.com/iced-rs/iced_aw)'s tab bar with additional features.

[![Watch the video](https://raw.githubusercontent.com/Fee0/iced_tabs/main/assets/thumbnail.png)](https://raw.githubusercontent.com/Fee0/iced_tabs/main/assets/video.mp4)

Features:

* Reordering by dragging
* Three Tab types: Text, Icon, Text + Icon
* Close buttons
* Tooltips
* Scrolling by scroll wheel or scrollbar
