Tab bar based on [iced_aw](https://github.com/iced-rs/iced_aw)

Features:

* Reordering by dragging
* Three Tab types: Text, Icon, Text + Icon
* Optional Close buttons
* Optional Tooltips
* Scrolling by scroll wheel
* Scrolling by scrollbar
